# The Life Junction

Your daily vibe station -- a curated Spotify playlist landing page.

## Live Site

Visit the site at: [https://canvickhush.github.io/The-Life-Junction/](https://canvickhush.github.io/The-Life-Junction/)

## About

The Life Junction is a simple, elegant landing page that links to a curated Spotify playlist. It features a glassmorphism UI design with a glowing title animation.

## Development

This is a static HTML site deployed via GitHub Pages. To run locally, simply open `index.html` in your browser.
